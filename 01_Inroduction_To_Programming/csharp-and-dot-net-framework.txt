C# is a modern, general-purpose, object-oriented, high-level programming language. The .NET platform contains the C# language, CLR (Common Language Runtime) and many auxiliary instruments and libraries ready for use. There are a few versions of .NET according to the targeted user group: NET Framework is the most common version of the .NET environment because of its general purpose. It is used in the development of console applications, Windows applications with a graphical user interface, web applications and many more.
